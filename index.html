# Google Sheets setup 1. Create a blank Google Sheet named **Kelly Drawing Contest**. 2. Copy the Sheet ID from its URL: `docs.google.com/spreadsheets/d/SHEET_ID/edit`. 3. In the Sheet, open **Extensions → Apps Script**. 4. Delete the sample code and paste `Code.gs`. 5. Replace `PASTE_YOUR_GOOGLE_SHEET_ID_HERE` with the Sheet ID. 6. Click **Deploy → New deployment → Web app**. 7. Set **Execute as: Me** and **Who has access: Anyone**. 8. Deploy, approve permissions, and copy the URL ending in `/exec`. 9. Open `index.html`, find `PASTE_YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE`, and replace it with that `/exec` URL. 10. Upload the updated `index.html` to GitHub Pages and submit one test entry. Each entry saves a row in Sheets and a private PNG in a Google Drive folder named **Kelly Drawing Contest Entries**. ## Important - Marketing consent is optional and separate from contest entry. - A parent/guardian should enter details for children under 13. - Add official rules stating the prize, eligibility, deadline, judging criteria, and how the winner is contacted. - Restrict Sheet and Drive access to authorized staff.
